## What is the GH-600 exam?

| | |
|---|---|
| **Full name** | GitHub Certified: Agentic AI Developer |
| **Exam code** | GH-600 |
| **Length** | 120 minutes |
| **Questions** | ~40–60 |
| **Passing score** | 700 / 1000 |
| **Status** | Beta (credential is identical to the GA exam) |
| **Audience** | Developers building agentic AI systems on GitHub |

It tests real skills: the software development lifecycle (SDLC), GitHub workflows and controls, code quality / security / review, and **coding agents** — GitHub Copilot, MCP (Model Context Protocol) servers, and agent customization.

**Official study guide:** [Microsoft Learn — Exam GH-600](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/gh-600) · [Certification page](https://learn.microsoft.com/en-us/credentials/certifications/agentic-ai-developer/)
